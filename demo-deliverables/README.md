# Hospital Management System - Demo Deliverables

This folder contains all the demo assets generated from the Cypress E2E test.

## Files

1. **cypress_video.mp4** - Video recording of the E2E test
2. **mouse_log.json** - Log of all mouse movements during the test
3. **click_log.json** - Log of all click events during the test
4. **demo_script.txt** - Natural language voiceover script for ElevenLabs
5. **demo_subtitles.srt** - SRT subtitle file for video editors

## How to Use

### Voiceover (ElevenLabs)
1. Open `demo_script.txt`
2. Copy the text to ElevenLabs
3. Generate voiceover with professional tone
4. Download the audio file

### Video Editing (CapCut or similar)
1. Import `cypress_video.mp4`
2. Import the generated voiceover audio
3. Import `demo_subtitles.srt` for subtitles
4. Sync and export final video

## Logs
The mouse and click logs can be used for:
- Analytics on user interaction patterns
- Heatmap generation
- Replay functionality
- Custom script generation
